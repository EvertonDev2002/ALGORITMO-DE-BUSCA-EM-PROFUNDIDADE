# Instruções e pré-requisitos
`Python` | `pip`
## Criar ambiente de desenvolvimento
Linux
```
$ python3 -m venv venv
```
ou
```
$ python -m venv venv
```
Windows
```
python -m venv venv
```

## Ativar ambiente de desenvolvimento
Linux
```
$ source venv/bin/activate
```
Windows
```
venv/Scripts/activate.bat
```

## Instalar dependências
```
$ pip install -r requirements.txt
```
